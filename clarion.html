About Clarion

A single file, offline, local first bench tool for reading, editing, and writing handheld radio memories over Web Serial. No server, no build step, no telemetry, no analytics. Nothing leaves the machine.

Licence. GPL-3.0.

Credits. The Baofeng 5RM memory layout and transfer protocol are derived from the CHIRP project's baofeng_uv17Pro driver by Sander van der Wel, GPL. The UV-5R clone protocol follows the long standing CHIRP uv5r driver. CHIRP is at chirpmyradio.com. No third party code is vendored into this file.

Embedded assets. One SVG icon and one manifest, both original work, both under the app's own licence, both inline as data URIs. No fonts, images, or audio are shipped. Asset budget: under 4 KB.

Offline. This file is the whole application. Save it and open it. The service worker layer only applies when the file is served over HTTP, since a service worker cannot be registered from a file URL. It is additive: nothing depends on it.

Feedback

Copies a plain text report to the clipboard. Paste it into an issue at github.com/mbparks/clarion. Nothing is sent anywhere by this app.
